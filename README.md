# Zeron V1 Protocol

This repository contains the core smart contracts for the Zeron V1 Protocol.


## Local deployment
Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.js
```
